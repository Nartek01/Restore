Sammanfattning:
Vad är .Net?
    .Net är en bibliotek eller en samling av funktioner,variabled, properties och methods. Som en utvecklare kan nyttja för att skapa webb applikationer, webbsidor, api serverar, programvara m.m
Vad är en API?
    Application Programming Interface (API) är ett sätt att förmedla data mellan två stycken platformar som annars inte kan förstår varandra. Likt hur fil formatet JSON fungerar för att förmedla subset och superset till andra applikationer. Faktumet är att API strukturen är formatterad i JSON.
Använda dotnet-CLIn
    Vi fick även lära oss att skapa ett API Projekt, koppla databasen med våran kodbas och köra våran projekt med dotnet-CLIn
Vad projekten innehåller
    Vi gick igenom hur "program" klassen fungerar, hur applikationen starts upp och vilka metoder att leta efter. Även modifierat till våran preferens.
